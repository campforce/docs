# How To Implement Features

# Set up feature branch
Click on `main` and write name for a new branch. Your branch should always have the same name as ticket, which you are working on.
<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179800895-76d6de53-fde2-402e-bac4-7b789b1b3649.png"></p><br>

## Fork
**Fetch to download contents from a remote repository** <br>
Press `Command + O` on Mac (`Ctrl + O` on Windows) and open `property-management` folder. Click to `Fetch button` on navigation bar.

<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179855658-e906f150-e3a2-4488-9ae1-8b07153eed10.png"></p><br>

**Move to new feature branch** <br>
After successful fetching you should see new remote repository on left sidebar. Right click on a new branch and checkout to move there.
<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179856736-0e3a2cc0-4e82-4f9f-9f2d-c7ba8940dc43.png"></p><br>

# Push
After implementation, check all your changes and adding. Green highlighted code is your adding, red highlighted code is your removing

<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179857911-9a4e66f7-deaa-4106-93a8-739ba2037281.png"></p><br>

Select all your files, you need to push and click on `Stage button`. Write a descriptive `Commit subject` and click `Commit button`

<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179858651-8742648f-ad58-4a47-b5eb-8a975c6c33c7.png"></p><br>

Don't forget to click to `Push button` on navigation bar.
<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179859100-d1e6d2ed-6f50-444c-9efb-e074478b82d5.png"></p><br>

# Pull-request
You should see notification on project's repository. Click to `Compare & pull-request` and create a [pull-request](https://www.atlassian.com/git/tutorials/making-a-pull-request). 
<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179859595-55e5cb35-ae24-4a67-a627-6972b62d9953.png"></p><br>

Give the same name as ticket for a pull-request title. Click to 'No one-assign yourself' text to assign yourself, then click to wheel right to Reviewers to assign reviewer for pull-request.
<p align="center"><img width="721" alt="command_palette" src="https://user-images.githubusercontent.com/89274213/179859913-93bc9220-a18e-42f8-870c-39bd4071b07c.png"></p><br>

## Merge
Wait for an approval from a reviewer before merging your changes into `main` branch.
<p align="center"><img width="910" alt="PR-REVIEW" src="https://user-images.githubusercontent.com/89274213/179863411-1bb06b4f-d1b1-4065-ab16-a0fc627e7188.png"></p><br>

Once you get approve you, merge your changes to `main` branch.
<p align="center"><img width="910" alt="PR-REVIEW" src="https://user-images.githubusercontent.com/89274213/180029516-9ecabd03-72ec-429c-b320-f82f930da285.png"></p><br>

Your changes are prepared to deploy. **Do not delete** your branch.
<p align="center"><img width="910" alt="PR-REVIEW" src="https://user-images.githubusercontent.com/89274213/180030215-0416cf50-cebf-4785-8465-a1dfc3e77e7b.png"></p><br>
